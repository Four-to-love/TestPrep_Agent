# SKILL.md

---
name: calendar_export
description: Generate iCalendar (.ics) files from the schedule DataFrame and provide a Streamlit download button.
---

## Overview
This skill provides a small Python helper that reads the `st.session_state.schedule_df` generated by the `GET_ANALYTICS` interceptor, converts each week entry into an `ics.Event`, and returns the path to a temporary `.ics` file. It also includes usage instructions for adding a download button in Streamlit.

## Files
- `export_ics.py` – implementation of `export_schedule_to_ics`.

## Usage in `app.py`
```python
from agents.skills.calendar_export.export_ics import export_schedule_to_ics

if not st.session_state.schedule_df.empty:
    ics_path = export_schedule_to_ics(st.session_state.schedule_df)
    with open(ics_path, "rb") as f:
        st.download_button(
            label="Download Schedule (ICS)",
            data=f,
            file_name="schedule.ics",
            mime="text/calendar"
        )
```

## Implementation Details (export_ics.py)
The helper creates a Calendar, adds an Event for each week (`Week X`), sets the start date based on the week number (today + 7*(week‑1) days), and writes the calendar to a temporary file under the project’s `tmp/` directory.

## Dependencies
- `ics` (added to `requirements.txt` by the installer).
